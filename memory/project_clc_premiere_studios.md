CLC Premiere Studios is a full-stack Expo (React Native + Web) app in `C:\Users\cnesmith\projects\clc-premiere-studios`.

**Why:** Crystal Lynn Creates wanted a Twitch-like streaming + video editing + social scheduling app branded for her content creation business.

**Stack:** Expo SDK 51, Expo Router v3, NativeWind v4 (Tailwind), Zustand, React Native SVG

**Key features built:**
- Stream Studio with camera/mic/screen recording + daily time limits by tier
- CapCut/Splice-style video editor (filters, speed, transitions, AI tools)
- 20-track royalty-free music library (pop/country/rap/R&B) + AI music creator
- Social media hub: YouTube, Instagram, X, Snapchat OAuth + post scheduling
- 60 SVG cartoon skins (30 retro 80s/90s, 30 anime)
- 3 subscription tiers: Free (3 min/day), Basic $4.99 (50 min, 1 post/skin/mo), Pro $10 (unlimited)
- Jade (#00A86B) color theme throughout

**Pending (production):** Node.js not installed on machine — user needs to install from nodejs.org. Social OAuth, Stripe payments, Mux streaming, AI APIs (Suno, Whisper, ElevenLabs) all need real API keys. Real logo PNG should replace SVG in assets/logo.png.

**How to apply:** When the user asks about this project, reference the SETUP.md for all integration details.
